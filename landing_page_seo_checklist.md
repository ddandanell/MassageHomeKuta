# Landing Page SEO Checklist
## Pre-Launch Verification for High-Converting Landing Pages

This checklist ensures every new landing page meets SEO best practices, conversion optimization standards, and technical requirements before going live.

## ‚úÖ ON-PAGE SEO ELEMENTS

### Meta Tags & Titles
- [ ] **Page Title** is 50-60 characters
- [ ] **Primary keyword** appears in page title
- [ ] **Meta description** is 145-155 characters
- [ ] **Meta description** includes primary keyword
- [ ] **Meta description** includes clear CTA
- [ ] **Title tag** is unique (not duplicate of other pages)
- [ ] **Canonical URL** is set correctly

### URL Structure
- [ ] **URL slug** is clean and readable
- [ ] **URL slug** includes primary keyword
- [ ] **URL slug** uses hyphens (not underscores)
- [ ] **URL slug** is lowercase
- [ ] **URL** is under 75 characters
- [ ] **No special characters** in URL (except hyphens)

### Heading Structure
- [ ] **One H1 tag** only
- [ ] **H1 includes primary keyword**
- [ ] **H2 tags** are used for main sections (5-8 H2s)
- [ ] **H3 tags** are used for subsections
- [ ] **Heading hierarchy** is logical (H1 > H2 > H3, no skipping)
- [ ] **Secondary keywords** appear in H2/H3 tags
- [ ] **Headings are descriptive** (not generic)

### Content Quality
- [ ] **Primary keyword density** is 1-2% (natural usage)
- [ ] **Secondary keywords** used naturally throughout
- [ ] **Content is 900-1200 words** minimum
- [ ] **First paragraph** includes primary keyword
- [ ] **Last paragraph** includes primary keyword
- [ ] **Content is original** (no duplicate content)
- [ ] **Content is scannable** (short paragraphs, bullet points)
- [ ] **Benefits-focused** (not just features)
- [ ] **Tourist-friendly language** (no jargon)
- [ ] **Local context included** (Bali-specific details)

---

## üîó INTERNAL LINKING

### Link Quantity
- [ ] **5-8 internal links** included
- [ ] **Links to related services**
- [ ] **Links to relevant area pages**
- [ ] **Links to main services/pricing pages**
- [ ] **No broken internal links**

### Link Quality
- [ ] **Anchor text is descriptive** (not "click here")
- [ ] **Anchor text includes keywords** where natural
- [ ] **Links are contextually relevant**
- [ ] **Links open in same tab** (for internal links)
- [ ] **Links are evenly distributed** throughout content

### Link Strategy
- [ ] **Links follow hierarchy** (general to specific)
- [ ] **Circular linking** (related pages link back)
- [ ] **Links support user journey**
- [ ] **Footer links** to key pages
- [ ] **Breadcrumb links** functional

## üèóÔ∏è TECHNICAL SEO

### Schema Markup
- [ ] **Service schema** implemented
- [ ] **LocalBusiness schema** on site
- [ ] **FAQPage schema** for FAQ section
- [ ] **AggregateRating schema** if reviews exist
- [ ] **Breadcrumb schema** implemented
- [ ] **Schema validates** in Google testing tool

### Page Speed
- [ ] **PageSpeed Insights score** > 80 (mobile)
- [ ] **PageSpeed Insights score** > 90 (desktop)
- [ ] **First Contentful Paint** < 1.8s
- [ ] **Largest Contentful Paint** < 2.5s
- [ ] **Time to Interactive** < 3.5s
- [ ] **Cumulative Layout Shift** < 0.1

### Technical Elements
- [ ] **HTTPS** enabled (secure)
- [ ] **Robots.txt** allows crawling
- [ ] **XML sitemap** includes page
- [ ] **No console errors**
- [ ] **No broken links** (404s)
- [ ] **Hreflang tags** if multilingual
- [ ] **Open Graph tags** for social sharing
- [ ] **Twitter Card tags** for Twitter

### Accessibility
- [ ] **Color contrast** meets WCAG AA standards
- [ ] **Keyboard navigation** works
- [ ] **Screen reader friendly**
- [ ] **ARIA labels** where needed
- [ ] **Focus indicators** visible
- [ ] **No auto-play media**
